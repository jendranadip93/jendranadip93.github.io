Kali


bertarung dalam jarak dekat (seduction) dengan menggunakan charm atau Fans. Pada level 15, karakter ini dapat dikhususkan untuk menjadi Dancer atau Screamer. Dancer pada umumnya bertarung secara Melee dan lebih banyak Physical Attack Based dibandingkan Magical Attack Based. Sedangkan Screamer pada umumnya bertarung secara Range dan lebih banyak menggunakan Magical Attack Based.
Pada level 45 terdapat 4 Class yang dapat dipilih yaitu Spirit Dancer dan Blade Dancer untuk Dancer,sedangkan Screamer yaitu Dark Summoner dan Soul Eater.
Ada satu special class kali, yaitu : Oracle Elder.


  1. Dancer


    2. Class yang mengutamakan daya serang jarak dekat. Dengan menggunakan cakram sebagai senjata utamanya.


    • Spirit Dancer


      • Class dari Dancer dengan skill yang mengeluarkan bantuan dari arwah leluhur nya.


    • Blade Dancer


      • Class dari Dancer dengan skill combo yang indah dan mematikan. Class ini berbeda dengan spirit dancer. Spirit dancer mengeluarkan arwah untuk membantunya menyerang. Kalau blade dancer, menggunakan kelenturan tubuhnya untuk menyerang.


  2. Screamer


    3. Class ini menggunakan kegelapan untuk menyerang musuhnya. Screamer sangat berguna ketika dalam party bersama teman. Karena kali mempunyai buff untuk membuat teman se-party nya menjadi kuat.


    • Dark Summoner


      • Class dari Screamer Mid Long Range attack yang memiliki skill regenerasi ketika menyerang. Dark summoner menggunakan kekuatan dari kegelapan yang ia dapat dari leluhurnya.


    • Soul Eater


      • Class dari Screamer Long Range Attack yang dapat mengeluarkan buff untuk membantu anggota party nya dan buff untuk mengalahkan musuh nya. Soul Eater menggunakan arwah - arwah leluhurnya untuk membantunya menyerang.


  3. Oracle Elder


    4. Lanjutan special class dari kali. Class ini menggunakan senjata cakram dan charm. Light attack dan serangan jarak dekat adalah andalannya.
